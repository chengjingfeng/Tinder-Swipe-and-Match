## Commit Log Convention

- 포멧 : `{prefix}: {변경 내용}`
- prefix
  - 기능 추가 : `feat:`
  - 인터페이스 추가 : `ui:`
  - 오류 수정 : `fix:`
  - 잡다한 일 : `chore`
  - 리팩토링 : `refactor:`
  - 파일추가 : `add:`
